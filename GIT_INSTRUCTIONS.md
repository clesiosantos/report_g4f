# Instruções de Clone (Ambiente REDUC)

Para clonar o repositório deste projeto baixando diretamente a branch de homologação da REDUC, utilize o comando abaixo no seu terminal local:

git clone -b REDUC https://github.com/clesiosantos/report_g4f.git

## Explicação dos parâmetros:
- **git clone**: Copia o repositório para sua máquina.
- **-b REDUC**: Especifica que você deseja baixar a branch chamada "REDUC".
- **URL**: O endereço do repositório no GitHub.
